Cool new DoS method check it out now!

The CISPA Helmholtz-Center for Information Security devised a cool new attack known as "Loop DoS", which works pretty much like you'd expect. It utilizes a vulnerability in the UDP protocol (CVE-2024-2169) making it susceptible to IP spoofing. This allows an attacked to send a single message to start a communication loop, which can cause outages.
